# Tocka pro výběr jmen

Jednoduchá webová aplikace pro náhodný výběr jmen pomocí otáčející se točky.

## Jak používat

1. Otevřete `index.html` v prohlížeči nebo spusťte server pomocí `python3 -m http.server 8000` a otevřete http://localhost:8000
2. Zadejte jména do pole a klikněte na "Přidat jméno"
3. Jména se zobrazí v seznamu a na točce
4. Klikněte na "Zatřást tockou" pro náhodný výběr
5. Vybrané jméno se zobrazí pod točkou
6. Jména můžete odebrat kliknutím na "Odebrat" vedle jména

## Funkce

- Přidávání a odebírání jmen
- Vizuální točka s barevnými segmenty
- Animovaná rotace točky
- Náhodný výběr jména
